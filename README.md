# wobbler
Keep the screen awake!

## Installation

1. Install [uv](https://docs.astral.sh/uv/getting-started/installation)
2. Run `uv sync` to set up the environment.
3. Use uv run <eavesdrop.py> to launch the program.

## Running

```bash
uv run python -m wobbler
```
